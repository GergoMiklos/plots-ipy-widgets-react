
plots_ipy_widgets_react
=====================================

Version: |release|

A Custom Jupyter Widget Library


Quickstart
----------

To get started with plots_ipy_widgets_react, install with pip::

    pip install plots_ipy_widgets_react

or with conda::

    conda install plots_ipy_widgets_react
